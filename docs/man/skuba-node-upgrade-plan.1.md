% skuba-node-upgrade-plan(1) # skuba node upgrade plan - Plan node upgrade

# NAME

plan - Print the upgrade plan of the node

# SYNOPSIS
**plan**
[**--help**|**-h**]
*plan* *<node-name>* [-h]

# DESCRIPTION
**plan** Evaluates and prints to stdout the upgrade plan for the given node

# OPTIONS

**--help, -h**
  Print usage statement.
